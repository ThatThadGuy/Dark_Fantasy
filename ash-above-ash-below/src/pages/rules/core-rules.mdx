# Core Loop
Roll **d20 + Attribute + Skill vs TN**. Natural 20 = critical; natural 1 = mishap.

## Attributes & Skills
- MGT (Might)
- AGL (Agility)
- WIT (Wit)
- RES (Resolve)
- GLM (Gloom)

## Advantage / Disadvantage
Roll 2d20, keep highest (adv) or lowest (dis).
